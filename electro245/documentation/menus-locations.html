Menus & Locations

Introduction

Electro has 10 menu locations and they are :

  • Off Canvas Menu – Appears on all headers except Header v6
  • Top Bar Left – Appears on all headers except Header v5
  • Top Bar Right – Appears on all headers except Header v5
  • Primary Nav – Appears on Header v2
  • Navbar Primary – Appears on Header v3
  • Secondary Nav – Appears on Header v1
  • Departments Menu – Appears on Header v2
  • All Departments Menu – Appears on Header v1
  • Blog Menu – Appears on Blog Page only
  • Mobile Handheld Department – Appears on Mobile Home v2 only

All the menus can be assigned from Appearance > Menus > Manage Location

Top Bar, All Departments & Secondary Nav

Header v1 has the following menu locations :

  • Hamburger Menu trigger for OffCanvas Menu
  • Top Bar Left
  • Top Bar Right
